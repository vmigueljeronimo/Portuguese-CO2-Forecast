📃 Introduction¶

The goal of this project is to develop a machine learning model that can forecast future CO2 emissions for Portugal. The model will be trained using Portuguese available data that has an impact on the CO2 emissions.

Additionally, the project will involve:

Data Collection & Preprocessing: Gathering a dataset with real data related to the topic.

Model Development: Implementing and training a model, optimizing the forecast accuracy.

Validation & Testing: Evaluating the model using real-world data.

User Interface: Designing a simple dashboard with the important information requested by the stakeholder.

import seaborn as sns

# Compute correlation matrix
corr_matrix = features_of_interest.corr()

# Plot the heatmap
plt.figure(figsize=(10, 6))
sns.heatmap(corr_matrix, annot=True, cmap='coolwarm', fmt=".2f", linewidths=0.5)
plt.title('Correlation Heatmap: Features vs CO₂ Emissions (Portugal 2005–2020)')

From the heatmap, we can conclude that the features most strongly correlated with CO₂ emissions in Portugal between 2005 and 2020 are gases_estufa (0.99), fossil_fuel_consumption (0.97), consumo_energia (0.96), and energia_primaria (0.94). These very high positive correlations indicate that as these variables increase, CO₂ emissions also tend to increase proportionally, making them excellent candidates for predicting future emissions. On the other hand, renewables_share_energy (-0.94) and produtividade_energia (-0.94) are strongly negatively correlated with CO₂, meaning that higher renewable energy usage and better energy productivity are linked to lower emissions. These features are also valuable for prediction, especially in scenarios focusing on mitigation. Variables like population (0.57) and primary_energy_consumption (0.35) show only moderate or weak correlations, and GDP (-0.11) appears almost unrelated, suggesting limited predictive power for CO₂ in this context.
